## tools

1.   wiki，[ctf wiki](https://ctf-wiki.org/)
2.   指令备忘录，[Quick Reference](https://reference.shanwer.top/)
3.   查提权利用 examples，[GTFOBins](https://gtfobins.github.io/)
4.   在线编码加解密工具，[cyberchef](https://icyberchef.com/)
5.   查询 glibc 符号表等信息，[glibc search](https://libc.rip/)
6.   问问题之前先看这个，[提問的智慧](https://github.com/ryanhanwu/How-To-Ask-Questions-The-Smart-Way.git)
7.   ip 信息查询，[iprss](https://ip.rss.ink/)
8.   shell 指令解释，[explainshell](https://www.explainshell.com/)
