# sandboxie

一款专为 Windows 平台设计的开源沙盒软件，能在操作系统上构建隔离的虚拟环境，让程序在其中安全运行，所有操作和更改都不会对主系统数据及设置产生永久性影响

官网：https://sandboxie-plus.com/

## install

1. 去官网下载 sandboxie-plus，安装时建议不要选择 *提取全部文件到指定目录供便携化使用*，非赞助者功能其实够用，其他按需选择。安装完成会在 C 下创建一个 sandbox 文件夹，这个文件夹将作为 sandboxie-plus 模拟出的资源管理器
2. 在 *选项 - 全局设置 - 高级设置* 中，修改 *沙盒文件系统根目录*
3. 在 sanboxie-plus manager 里，可以新建几个 sanboxie 用于不同用途
4. 以安装微信为例，在官网下载安装包后，右键安装包选择 "在沙盒中运行"，即可使用沙盒的上下文来实现与主系统隔离
